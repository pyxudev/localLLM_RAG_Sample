# Apex

**特徴**:

- **Salesforce 専用言語**: Salesforce プラットフォームで動作
- **ローコード**: 可視化されたプログラミング
- **非同期処理**: バッチ処理に対応
- **テスト性**: カバレッジ要件が厳格
- **外部連携**: REST/SOAP API 統合

## オブジェクト指向

**重要概念**:

- **カプセル化**: データと処理をまとめる
- **継承**: 既存クラスを拡張
- **ポリモーフィズム**: 同じインターフェースで異なる実装
- **アクセス修飾子**: `public`, `protected`, `private`
